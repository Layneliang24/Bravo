# 规则应用声明模板

## 统一声明格式

所有规则文件必须在文件末尾添加以下声明部分：

```markdown
---

## 规则应用声明

**当AI被路由到此规则时，必须在响应开头明确声明**：
```

[应用规则：@.cursor/rules/{规则路径}]
[切换到{角色名称}角色]（如果有角色要求）

```

**验证要求**：
- ✅ 必须明确声明应用了此规则
- ✅ 必须遵循本规则中的所有约束和流程
- ✅ 如果本规则定义了角色，必须切换到对应角色
- ✅ 不能跳过声明步骤直接执行操作
```

## 使用说明

1. **规则路径**：使用完整的相对路径，如 `@.cursor/rules/01-product/prd-standards.mdc`
2. **角色名称**：如果规则定义了角色（如"PRD设计专家"），必须声明角色切换
3. **位置**：声明部分必须放在规则文件的最末尾
4. **格式**：保持统一的格式，便于AI识别和遵循
